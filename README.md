# Procurement Savings Estimator

A simple interactive calculator that estimates potential procurement value leakage in enterprise supply chains.

The tool helps procurement teams understand how much value may be lost due to supplier complexity, manual processes, and contract inconsistencies.

## How it works

Users answer a few quick questions about:

- Annual procurement spend
- Number of suppliers
- ERP system
- Procurement automation level

The tool then estimates potential savings based on common procurement leakage benchmarks.

## Why this tool exists

This tool was built as a small growth experiment inspired by how companies like Magentic help enterprises detect and capture procurement savings using AI agents.

## Tech Stack

- Python
- Streamlit
